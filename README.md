# W3D2 - CRUD in Express

## CRUD

* Create
* Read
* Update
* Delete

## `dogs` resource

### Read

#### index

* `GET /dogs`

#### show

* `GET /dogs/:id`

### Create

#### new

* `GET /dogs/new`

#### create

* `POST /dogs`

### Update

#### edit

* `GET /dogs/:id/edit`

#### update

* `PUT /dogs/:id`

w/ Forms

* `POST /dogs/:id`

### Delete

#### delete

* `DELETE /dogs/:id`

w/ Forms

* `POST /dogs/:id/delete`
